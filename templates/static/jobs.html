Jobs at Sefaria משרות פנויות בספאריה

Junior Content Engineer

עברית בהמשך

Sefaria, the open digital platform for Jewish texts, seeks a driven individual to help expand our library of text. This full-time, junior programmer will work on content-related tasks for our system. The role involves:

  • Analyzing structures of texts and creating formal representations using Sefaria's toolset
  • Parsing text from various sources and loading them into Sefaria’s repository
  • Developing and improving tools used in content processing
  • Developing and improving algorithms used to find connections between different texts
  • Creating tools for non-technical power users to edit and manage large volumes of text

Key Responsibilities:

Sefaria’s database currently contains more than 60 million words. We’re working through a roadmap to triple the size of the library by the end of 2018. Everything we obtain must be properly imported into our system. Our repository’s code base is Python running over MongoDB with a RESTful API. The process of parsing unstructured digital text generally involves analyzing the text to determine its structure, encoding that structure, writing custom scripts to convert the source text into the appropriate structure, and posting the resulting data to our server.


Desired Experience and Skills: Applicants need to understand the structure of Torah literature (http://www.sefaria.org/texts), and should have a background in programming and a history of learning Torah.
  • At least one year of programming experience
  • Python (or comparable experience)
  • Regular expressions and fuzzy matching
  • Knowledge of Judaic text (e.g., Torah, Talmud, Kabbalistic works, halakhic works, commentaries, etc.)
  • Understanding of written Hebrew
  • English (professional working proficiency)
  • Detail oriented
  • Proactive worker and a self-starter

Location: Jerusalem.

Job Type: This is a full time permanent position.

To apply, please send your resume to jobs@sefaria.org. If you have a GitHub account, please include a link on your resume.

מתכנת תוכן זוטר

ספאריה, הפלטפורמה הפתוחה לטקסטים יהודיים, מחפשת מישהו או מישהי עם מוטיבציה שיעזרו להרחיב את ספריית הטקסטים שלנו. מדובר בתפקיד משרת מתכנת זוטר במשרה מלאה שיעסוק בתוכן עבור המערכת שלנו. התפקיד כולל:

  • אנליזה של מבנה הטקסט ויצירת ייצוגים פורמליים באמצעות סט הכלים שלנו.
  • כריית טקסטים ממקורות שונים למרכיביהם והטענתם למאגר של ספאריה.
  • פיתוח ושיפור כלים לעיבוד תוכן.
  • פיתוח ושיפור אלגוריתמים למציאת קשרים בין טקסטים שונים.
  • יצירת כלים למשתמשים לא-טכניים לעריכה ולניהול כמות גדולה של טקסט.

תחומי אחריות:

מסד הנתונים של ספאריה כרגע מכיל יותר משישים מליון מילים. אנו כרגע עובדים לקראת הכפלת גודל הספריה פי שלוש עד סוף 2018. צריך להזין למאגר את כל החומרים שאנו משיגים בצורה הנכונה. המאגר שלנו כתוב בפייתון ורץ על MongoDB עם RESTful API. תהליך כריית הטקסט הדיגיטלי כרוך באנליזה של הטקסט כדי לגלות את המבנה שלו, קידוד המבנה, כתיבת סקריפטים להמרת טקסט המקור למבנה המיועד, והעלאת התוצאה לשרת שלנו.


ניסיון וכישורים דרושים: המועמדים צריכים להבין את המבנה של טקסטים מהתורה (http://www.sefaria.org/texts), וצריכים רקע בתכנות ובלימוד תורה.

  • ידע בטקסטים יהודיים (למשל תורה, תלמוד, קבלה, הלכה, מפרשים ועוד).
  • יכולת קריאה בעברית מן המקורות.
  • אנגלית ברמת שפת אם
